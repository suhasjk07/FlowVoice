## ğŸ› Troubleshooting Quick Fixes

### "npm is not recognized"
**Fix**: Restart your terminal after installing Node.js

### "git is not recognized"
**Fix**: Restart your terminal after installing Git

### Build fails locally
**Fix**: 
```bash
rm -rf node_modules package-lock.json
npm install
npm run build
```

### Can't push to GitHub
**Fix**:
```bash
git remote -v  # Check if remote is correct
git remote remove origin
git remote add origin YOUR_CORRECT_URL
git push -u origin main
```

### Vercel deployment fails
**Fix**: Check the build logs in Vercel dashboard
- Usually a missing dependency or build error
- Make sure `npm run build` works locally first
